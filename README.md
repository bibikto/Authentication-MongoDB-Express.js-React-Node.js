# Authentication - Node.js DynamoDB React

Node.js used for Backend.
DynamoDB(AWS) for storing user data.
React for frontend.

## Config Variables
In config/config.js:

- PORTS : Port used for http followed by port used for https. Ex :[80,443]
- KEYFILE
- CERTFILE

## Usage

```Node.js
npm start
```
